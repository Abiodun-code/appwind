## Run Locally

Clone the project

```bash
  git clone https://github.com/Abiodun-code/appwind
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn add
```

Start the server

```bash
  yarn dev
```


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
